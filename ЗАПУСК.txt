🚀 БЫСТРЫЙ ЗАПУСК (ОДИН КЛИК):

   Windows: Дважды кликните на файл "start.bat"
   
   Или откройте терминал и выполните:
   python start.py


📋 ЧТО НУЖНО ПЕРЕД ЗАПУСКОМ:

   1. Python 3.8+ (https://www.python.org/downloads/)
   2. Node.js 16+ (https://nodejs.org/)
   3. LM Studio (https://lmstudio.ai/)


⚙️ НАСТРОЙКА LM STUDIO:

   1. Откройте LM Studio
   2. Загрузите модель (~7B параметров)
   3. Перейдите в "Local Server"
   4. Нажмите "Start Server"
   5. Убедитесь, что порт 1234 активен


✨ ЧТО ДЕЛАЕТ СКРИПТ:

   ✓ Проверяет наличие Python и Node.js
   ✓ Создает виртуальное окружение
   ✓ Устанавливает все зависимости автоматически
   ✓ Запускает backend (порт 8000)
   ✓ Запускает frontend (порт 5173)
   ✓ Открывает браузер с приложением


🌐 ПОСЛЕ ЗАПУСКА:

   Приложение откроется автоматически на http://localhost:5173
   
   Если не открылось - откройте вручную:
   http://localhost:5173


🛑 ОСТАНОВКА:

   Нажмите Ctrl+C в терминале


❓ ПРОБЛЕМЫ?

   - Python не найден → Установите Python и отметьте "Add to PATH"
   - Node.js не найден → Установите Node.js
   - Порты заняты → Закройте другие приложения на портах 8000/5173
   - LM Studio не работает → Приложение запустится, но без генерации
